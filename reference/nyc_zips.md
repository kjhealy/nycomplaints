# NYC Zip Codes by borough

Zip codes and borough names

## Usage

``` r
nyc_zips
```

## Format

A data frame with 211 rows and 5 variables:

- `zip` Zip code (char)

- `borough` Borough name

- `city` City name (USPS)

- `county` County name

- `long_county` County name, New York

- `short_county` One word county name

## Details

Sources:
https://www.nyc.gov/assets/planning/download/office/data-maps/nyc-population/census2000/sf1p11.xls
and https://worldpopulationreview.com/zips/new-york
